# AI  spam detection analyzer - Spam Detection Web Application

## 📋 Project Overview
- **Model**: Naive Bayes (Text Classification)
- **Dataset**: SMS Spam Collection (`sms.tsv`)
- **Backend**: Flask (Python)
- **Frontend**: HTML + CSS
- **Features**: Real-time prediction, Threading, Unit Testing, Postman API Testing

## ✨ Features Implemented
- ✅ Single clean web interface
- ✅ Spam/Ham prediction with confidence score
- ✅ Threading for non-blocking prediction
- ✅ Unit testing using `unittest`
- ✅ API testing with Postman
- ✅ Error handling
- ✅ Responsive design

## 🛠 Technologies Used
- Python, Flask, scikit-learn, pandas
- HTML, CSS, JavaScript
- unittest framework
